## 三种运作模式

### 模式一：完整论文模式
```
输入: "计算机视觉 + 小样本学习"

流程: 曹操定调 → 荀彧规划 → 郭嘉创新 → 荀攸文献 → 
      程昱实验 → 贾诩分析 → 司马懿撰写 → 鲁肃校对

输出: 完整论文(含实验代码)
```

### 模式二：探索模式
```
输入: "量子计算"

流程: 曹操定调 → 荀彧规划 → 郭嘉创新 → 荀攸文献

输出: 3-5个潜在研究方向 + 各自优劣分析
```

### 模式三：专项模式
```
输入: "设计一个图像分类消融实验"

流程: 曹操授权 → 程昱设计 → 贾诩验证

输出: 完整实验方案 + 代码 + 分析报告
```
