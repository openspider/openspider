## 一、计划概述

### 1.1 计划命名

**OpenSpider计划** - 开放式科研生态网络

| 维度 | 含义 |
|------|------|
| **Open** | 开放、包容、连接一切可连接的资源 |
| **Spider** | 蜘蛛网式协同网络，连接四向、贯通南北 |
| **计划** | 系统化、组织化、持续迭代 |

**核心理念**: 像蜘蛛编织网络一样，NRT团队将构建一个开放、协同、智能的组织生态，连接科研、产业、资本、人才，形成强大的价值网络。

### 1.2 组织治理架构

    ┌─────────┐┌─────────┐┌─────────┐┌─────────┐┌─────────┐
    │东向负责人││西向负责人││南向负责人││北向负责人││       │

### 1.3 责权体系

| 层级 | 角色 | 职责 | 权限 |
|------|------|------|------|
| **L1** | 董事长 | 战略方向、重大决策、资源配置 | 最终决策权 |
| **L2** | 总裁（曹操） | 居中指挥、四向协调、整体管理 | 执行决策权 |
| **L3** | 四向负责人 | 各向执行、团队管理、结果交付 | 日常决策权 |
| **L4** | 各组组长 | 专业执行、任务分解、成果交付 | 操作执行权 |

### 1.4 决策流程

```
┌─────────────────────────────────────────────────────────────────┐
│                        决策流程                                   │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  日常事务: 四向负责人决策 → 执行                                 │
│      ↓                                                           │
│  重要事务: 四向负责人提案 → 总裁审核 → 执行                      │
│      ↓                                                           │
│  重大事务: 四向负责人提案 → 总裁审议 → 董事长裁决 → 执行         │
│      ↓                                                           │
│  战略事务: 总裁战略提案 → 董事长战略评议 → 战略会决议 → 执行    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

**犹豫不决处理原则**:
1. 涉及方向性问题 → 汇报董事长
2. 涉及资源配置冲突 → 曹操协调，重大报董事长
3. 涉及跨向协作 → 曹操协调仲裁
4. 涉及外部合作 → 曹操审批，重大报董事长

## 三、曹操（总裁）居中指挥机制

### 3.1 曹操的核心定位

**"居中指挥，四向联动"**

| 维度 | 职责 |
|------|------|
| **战略执行** | 将董事长的战略方向转化为执行方案 |
| **四向协调** | 协调东、西、南、北四向协同 |
| **资源调配** | 在四向间调配人、财、物资源 |
| **决策仲裁** | 对四向间的决策冲突进行仲裁 |
| **重大汇报** | 对犹豫不决之事向董事长汇报 |

### 3.2 日常管理机制

**日机制**:
| 时间 | 工作 | 输出 |
|------|------|------|
| 8:00 | 查看四向日报 | 进度清单 |
| 9:00 | 四向晨会（站会） | 协调事项 |
| 10:00 | 重点事项处理 | 决策指令 |
| 14:00 | 外部会议/拜访 | 合作推进 |
| 16:00 | 进度跟踪 | 问题清单 |
| 18:00 | 日报汇总 | 董事长汇报 |

**周机制**:
| 时间 | 工作 | 输出 |
|------|------|------|
| 周一 | 四向周例会 | 周计划 |
| 周三 | 重点突破会议 | 专项方案 |
| 周五 | 周度总结 | 周报 |

**月机制**:
| 时间 | 工作 | 输出 |
|------|------|------|
| 第1周 | 月度计划 | 月度OKR |
| 第2周 | 执行跟踪 | 进度报告 |
| 第3周 | 问题解决 | 解决方案 |
| 第4周 | 月度复盘 | 月度报告 |

### 3.3 资源调配机制

**资源池**:
| 资源类型 | 说明 | 调配原则 |
|----------|------|----------|
| 人力资源 | 各向人员调配 | 优先满足战略重点 |
| 财力资源 | 预算分配 | 投入产出比优先 |
| 客户资源 | 央企客户分配 | 能力匹配优先 |
| 技术资源 | 算法、平台共享 | 复用最大化 |

**调配流程**:
```
需求提出 → 资源评估 → 曹操审批 → 调配执行 → 效果评估
     ↓           ↓           ↓           ↓          ↓
  各向负责人   辛评/唐咨     曹操       相关向     曹操
```

### 3.4 决策仲裁机制

**仲裁事项**:
| 事项类型 | 仲裁人 | 重大报董事长 |
|----------|--------|--------------|
| 跨向资源冲突 | 曹操 | >50万资金 |
| 跨向优先级分歧 | 曹操 | 战略方向相关 |
| 外部合作归属 | 曹操 | >100万合同 |
| 重大风险判断 | 曹操 | 涉及公司存亡 |

**仲裁流程**:
```
争议提出 → 事实调查 → 曹操听证 → 裁决决策 → 执行跟踪
     ↓           ↓           ↓           ↓          ↓
  相关向      荀攸/情报组    曹操       曹操       曹操
```

### 3.5 重大事项汇报机制

**汇报标准**:
| 汇报类型 | 触发条件 | 汇报方式 |
|----------|----------|----------|
| 战略方向 | 重大战略调整建议 | 书面+面谈 |
| 资源配置 | 单项投入>100万 | 书面+面谈 |
| 外部合作 | 合作金额>100万 | 书面+面谈 |
| 重大风险 | 可能影响公司存亡 | 即时汇报 |
| 人员变动 | 核心岗位变动 | 书面汇报 |

**汇报格式**:
```
标题: [汇报类型] + 核心问题
背景: 事件背景简述
现状: 当前情况分析
问题: 犹豫不决之处
建议: 曹操建议方案
请董事长裁决事项:
1. xxx
2. xxx
```
