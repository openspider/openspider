## 一、总览

### 1.1 NRT团队定位

**NRT科研团队** = 数字科学家团队 × 国资央企 × 前沿科技

| 维度 | 定位 |
|------|------|
| **使命** | 以前沿科技服务国资央企数字化转型 |
| **愿景** | 成为数字科学家团队标杆，引领科研与产业融合 |
| **核心价值** | 前沿科研×深度产业理解×可持续商业模式 |

### 1.2 东西南北四向总览

| 方位 | 核心职能 | 战略使命 | 核心产出 |
|------|----------|----------|----------|
| **东向** | 行政 + 财务 | 组织运行的"供血系统" | 制度、预算、人才 |
| **西向** | 战略 + 科普 + 影响力 | 方向引领的"大脑中枢" | 战略、科普、品牌 |
| **南向** | 科研课题 | 知识生产的"创新引擎" | 论文、技术、专利 |
| **北向** | 业务变现 + 产业化 | 价值转化的"动力系统" | 产品、客户、收入 |

        └─────────────────────────────┼─────────────────────────────┘

### 2.2 协同原则

| 协同维度 | 流向 | 关键节点 | 责任方 |
|----------|------|----------|--------|
| **战略→科研** | 西向→南向 | 方向指引、课题设定 | 战略组→科研组 |
| **科研→产业** | 南向→北向 | 技术转让、产品化 | 科研组→产业组 |
| **产业→行政** | 北向→东向 | 财务回流、预算分配 | 产业组→法务组 |
| **行政→战略** | 东向→西向 | 资源汇报、组织健康度 | 法务组→战略组 |
| **科普→影响力** | 西向内部 | 内容输出、品牌建设 | 文化组 |
| **情报→全向** | 中央→四向 | 趋势洞察、风险预警 | 情报组 |

## 三、东向详细策划

### 3.1 定位与使命

**东向 = NRT的"供血系统" + "组织基石"**

| 维度 | 内涵 |
|------|------|
| **供血系统** | 财务管理、资金调配、预算控制 |
| **组织基石** | 制度建设、人员管理、行政运营 |

**核心使命**:
1. 保障组织运转顺畅
2. 实现资源最优配置
3. 构建现代治理体系
4. 防范运营风险

### 3.2 组织架构

```
┌─────────────────────────────────────────┐
│              东向 (荀彧 · COO)          │
├─────────────────────────────────────────┤
│                                         │
│  ┌──────────┐  ┌──────────┐  ┌───────┐ │
│  │  行政部  │  │  财务部  │  │ 人力部│ │
│  │ 鲁肃/费祎│  │ 辛评/唐咨│  │ 王观  │ │
│  └──────────┘  └──────────┘  └───────┘ │
│                                         │
└─────────────────────────────────────────┘
```

### 3.3 部门职能

| 部门 | 负责人 | 核心职能 |
|------|--------|----------|
| **行政部** | 鲁肃/费祎 | 制度建设、会议组织、文档管理、后勤保障 |
| **财务部** | 辛评/唐咨 | 预算管理、财务核算、资金管理、投融资 |
| **人力部** | 王观 | 招聘管理、绩效考核、培训发展、员工关系 |

### 3.5 预算分配模型

```
总预算 = 收入（业务收入+投资） + 存量资金

分配原则:
  科研投入 (40%)  ← 南向
    · 课题研究: 25%
    · 设备采购: 10%
    · 学术交流: 5%
  
  行政运营 (20%)  ← 东向
    · 人力成本: 12%
    · 办公费用: 5%
    · 管理系统: 3%
  
  产业发展 (25%)  ← 北向
    · 市场拓展: 10%
    · 产品研发: 10%
    · 品牌建设: 5%
  
  战略储备 (15%)
    · 应急资金: 10%
    · 机会投资: 5%
```

### 3.6 考核指标

| 指标 | 目标值 | 责任方 | 考核频率 |
|------|--------|--------|----------|
| 流程时效达标率 | ≥95% | 行政部 | 月度 |
| 预算执行准确率 | ±5% | 财务部 | 季度 |
| 招聘到岗率 | ≥90% | 人力部 | 季度 |
| 考核满意度 | ≥85% | 人力部 | 季度 |
| 员工满意度 | ≥80% | 人力部 | 年度 |
| 合规达标率 | 100% | 全部门 | 月度 |
