## 👥 四大科学家类型

### 1. 苏柏亚AI科学家 🧠

| 属性 | 值 |
|------|-----|
| **定位** | 核心科学家，整合古典智慧与AI |
| **核心能力** | 批判性思维、形式论、逻辑伦理 |
| **使命** | 用古典智慧照亮AI之路 |

### 2. 产业科学家 🏭

| 属性 | 值 |
|------|-----|
| **定位** | 产业价值创造 |
| **核心能力** | 市场分析、商业模式、技术转化 |
| **使命** | 将AI能力转化为产业价值 |

### 3. 学术科学家 🎓

| 属性 | 值 |
|------|-----|
| **定位** | 学术创新、论文发表 |
| **核心能力** | 文献分析、假设生成、论文写作 |
| **使命** | 追求学术卓越 |

### 4. 科研科学家 🔬

| 属性 | 值 |
|------|-----|
| **定位** | 基础研究、技术突破 |
| **核心能力** | 实验设计、技术评估、成果转化 |
| **使命** | 推动科学边界 |
