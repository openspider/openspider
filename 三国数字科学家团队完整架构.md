## 考核体系

### 团队层面考核

| 指标 | 目标值 | 责任组 |
|------|--------|--------|
| 战略执行到位率 | ≥90% | 战略组 |
| 风险预警准确率 | ≥85% | 谋略组 |
| 系统可用性 | ≥99.9% | 工程组 |
| 前沿论文复现率 | ≥70% | 科研组 |
| 收入增长率 | ≥20%/月 | 产业组 |
| 合规达标率 | 100% | 法务组 |
| 情报覆盖率 | ≥90% | 情报组 |
| 月度内容产出 | 20篇+ | 文化组 |

### 考核周期

| 考核类型 | 频率 | 责任人 |
|----------|------|--------|
| 实时考核 | 实时 | 各组监控负责人 |
| 周度考核 | 每周 | 各组组长 |
| 月度考核 | 每月 | 战略组汇总 |
| 季度考核 | 每季 | 诸葛亮主持 |
